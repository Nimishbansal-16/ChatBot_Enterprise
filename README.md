# ChatBot

## Getting started

## Name
Enterprise ChatBot

## Description
This project focus on enterprise chatbot, which will enable user to create chatbot and flows with no code.

## Project Setup
```commandline
cd ~/path/to/project
git clone https://git.ixigo.com/cxp/tara_enterprise.git
cd tara_enterprise
```

## Installation
Download python 3.11.4
https://www.python.org/downloads/release/python-3114/

## Create virtual environment
```commandline
python3.11 -m venv ../venv_te
```
-  venv_te refers to  <b>Virtual Environment Tara Enterprise</b>

## Package Support
```
pip install -r requirements.txt
```
